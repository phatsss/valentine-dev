## 📦 **Installation**

Clone the repo and install dependencies:

```sh
git clone https://github.com/phatsss/valentine-dev.git
cd valentine-dev
npm install
```

or using Yarn:

```sh
yarn install
```

---

## 🌈 **How to Run**

Start the project locally with:

```sh
npm run dev
```

_Open your browser and navigate to **\`http://localhost:5173/\`** (For Vite projects)._
